# BIST_Portfoy
BIST hisselerini gerçek zamanlı takip eden Android uygulaması. Kotlin ve MVVM mimarisiyle geliştirildi; anlık fiyat, günlük değişim ve maliyete göre kâr/zarar hesaplar. Foreground Service ile uygulama kapalıyken bile fiyat değişimlerinde push bildirimi gönderir. Halka arzlarda tahmini karşılanacak lot sayısını hesaplayan özel bir araç da içerir.
